Grid Parallax

3 Columns

Multi-row

JavaScript Options

Option Value Default Description
margin String x-grid-margin This class is added to items that break into the next row, typically to create margin to the previous row.
first-column String x-first-column This class is added to the first element in each row.
masonry Boolean false Enables masonry layout for this grid.
parallax Number 0 Parallax translation value. The value must be a positive integer. Falsy disables the parallax effect (default).
